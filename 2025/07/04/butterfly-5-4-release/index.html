Butterfly 5.4 Release Notes

New Features

  • Update dependency versions to latest in plugins.yml
  • Support for Fancybox v6
  • Add configuration option to display article title in navigation bar
  • Add hover tooltip to show “Back to Home” text in article page navigation bar
  • Add Google Tag Manager configuration support
  • Add animation effects for right-side configuration button, update related configurations and template logic
  • Update footer configuration, add navigation bar and copyright information, improve styles and structure

Improvements

  • Enhanced user interface responsiveness
  • Better mobile experience
  • Optimized loading performance
  • Improved accessibility features

Bug Fixes

  • Fixed various minor issues
  • Improved compatibility with latest Hexo versions
  • Enhanced stability across different browsers

Migration Guide

To upgrade to Butterfly 5.4:

  1. Backup your current configuration
  2. Update the theme files
  3. Review and update your _config.yml if needed
  4. Test your site thoroughly

Breaking Changes

Please refer to the documentation for any breaking changes that might affect your current setup.

What’s Next

We’re continuously working on improving Butterfly theme. Stay tuned for more exciting features in upcoming releases!

Thank you for using Butterfly theme!
